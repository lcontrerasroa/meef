  • Graphophonémique de l'anglais
    Correspondances graphème → phonème pour les voyelles accentuées
    • Monographes ‹a, e, i, o, u›
      5 voyelles graphiques × 4 contextes
      • Entravé (checked) — ˈVC# / ˈVCC
        Syllabe fermée : la voyelle est "coupée" par la coda consonantique
        • ‹a› → /æ/
          cat /kæt/, cap /kæp/, attack /əˈtæk/
        • ‹e› → /e/
          pet /pet/, spent /spent/, dress /dres/
        • ‹i› → /ɪ/
          bid /bɪd/, kit /kɪt/, twist /twɪst/
        • ‹o› → /ɒ/
          dot /dɒt/, lot /lɒt/, adopt /əˈdɒpt/
        • ‹u› → /ʌ/
          cut /kʌt/, strut /strʌt/, bus /bʌs/
      • Libre (free) — ˈVCe# / ˈV#
        Syllabe ouverte ou patron "magic e" : la voyelle résonne librement
        • ‹a› → /eɪ/
          Kate /keɪt/, make /meɪk/, late /leɪt/, name /neɪm/
        • ‹e› → /iː/
          Pete /piːt/, these /ðiːz/, me /miː/, we /wiː/
        • ‹i› → /aɪ/
          bide /baɪd/, time /taɪm/, my /maɪ/, try /traɪ/
        • ‹o› → /əʊ/
          note /nəʊt/, home /həʊm/, go /gəʊ/, hero /ˈhɪərəʊ/
        • ‹u› → /(j)uː/
          mute /mjuːt/, cute /kjuːt/, flute /fluːt/, use /juːz/
      • Entravé + R — ˈVr# / ˈVrC
        ‹r› muet en SSBE, mais modifie la voyelle → voyelles longues
        • ‹ar› → /ɑː/
          car /kɑː/, card /kɑːd/, start /stɑːt/, garden /ˈgɑːdən/
          Passage /æ/ → /ɑː/ : recul du point d'articulation
        • ‹er› → /ɜː/
          her /hɜː/, term /tɜːm/, verb /vɜːb/, perfect /ˈpɜːfəkt/
        • ‹ir› → /ɜː/
          bird /bɜːd/, girl /gɜːl/, first /fɜːst/, dirty /ˈdɜːti/
        • ‹or› → /ɔː/
          or /ɔː/, fork /fɔːk/, sport /spɔːt/, normal /ˈnɔːməl/
        • ‹ur› → /ɜː/
          burn /bɜːn/, fur /fɜː/, turn /tɜːn/, purple /ˈpɜːpəl/
        🔑 Convergence : ‹er›, ‹ir›, ‹ur› entravés → tous /ɜː/ (neutralisation).
        ⚠️ ‹-rr-› (double r) = pas de modification : carry /ˈkæri/, merry /ˈmeri/, sorry /ˈsɒri/
      • Libre + R — ˈVre#
        Voyelle libre + ‹r› → diphtongues centralisantes (sauf ‹o›)
        • ‹are› → /eə/
          care /keə/, bare /beə/, share /ʃeə/, stare /steə/
          Diphtongue centralisante : /e/ → /ə/
        • ‹ere› → /ɪə/
          here /hɪə/, mere /mɪə/, severe /sɪˈvɪə/, sincere /sɪnˈsɪə/
          Diphtongue centralisante : /ɪ/ → /ə/. Évolution : /ɪə/ → /ɪː/ chez beaucoup de locuteurs.
        • ‹ire› → /aɪə/
          fire /faɪə/, tire /taɪə/, wire /waɪə/, admire /ədˈmaɪə/
          Triphtongue : /a/ → /ɪ/ → /ə/
        • ‹ore› → /ɔː/
          more /mɔː/, bore /bɔː/, store /stɔː/, explore /ɪkˈsplɔː/
          Exception : monophtongue (pas de diphtongue). Identique à ‹or› entravé.
        • ‹ure› → /(j)ʊə/
          pure /pjʊə/, cure /kjʊə/, sure /ʃʊə/, mature /məˈtjʊə/
          Diphtongue centralisante : /ʊ/ → /ə/. Évolution : /ʊə/ → /ɔː/ chez beaucoup de locuteurs.
    • Sous-règles et exceptions
      Régularités qui "écrasent" les règles de base des monographes
      • 1) ‹a›/‹o› + ‹l› + C
        • 1A) ‹a› + ‹l› + {l#, t, d, k} → /ɔː/
          mall /mɔːl/, talk /tɔːk/, waltz /wɔːls/, bald /bɔːld/, Baltic /ˈbɔːltɪk/
          GA : /ɑː/ à la place de /ɔː/. NB : ‹l› muet devant ‹k› : walk, talk, folk…
        • 1B) ‹a› + ‹l› + {m, f} → /ɑː/
          half /hɑːf/, calm /kɑːm/, calf /kɑːf/, balm /bɑːm/
        • 1C) ‹o› + ‹l› + {l#, t, d, s, k} → /əʊ/
          roll /rəʊl/, bolt /bəʊlt/, told /təʊld/, gold /gəʊld/, folk /fəʊk/, most /məʊst/, host /həʊst/, post /pəʊst/
      • 2) "W rule" : /w/ + ‹a› et /w/ + ‹ar›/‹or›
        • 2A) ‹w› + ‹a› → /wɒ/ (SSBE)
          water /ˈwɒtə/, want /wɒnt/, quality /ˈkwɒlɪti/, swan /swɒn/
          Exception : si C suivante est vélaire /k, g, ŋ/ → retour à /æ/ : quack /kwæk/, wax /wæks/, wag /wæg/
        • 2B) ‹w› + ‹ar› → /wɔː/
          wart /wɔːt/, warm /wɔːm/, swarm /swɔːm/, quarter /ˈkwɔːtə/
        • 2C) ‹w› + ‹or› → /wɜː/
          word /wɜːd/, work /wɜːk/, world /wɜːld/, worm /wɜːm/, worse /wɜːs/
      • 3) Terminaisons "spéciales" → valeur libre
        La voyelle prend sa valeur libre au lieu d'entravée
        • 3A) -ange# → ‹a› = /eɪ/
          arrange, change, derange, range, strange ; aussi angel, danger
        • 3B) -aste# → ‹a› = /eɪ/
          baste, chaste, haste, paste, taste, waste
        • 3C) -igh(t)# → ‹i› = /aɪ/
          high, sigh, bright, tight, fight, flight, knight, righteous
        • 3D) -ign# → ‹i› = /aɪ/
          align, sign, resign, assign, benign, malign
        • 3E) -ind# → ‹i› = /aɪ/
          bind, blind, kind, find, grind, mind ; aussi child, climb, mild, pint, wild
      • 4) "Pull rule" : ‹f/p/b/m› + ‹u› + (-ll/-sh/-tch) → /ʊ/
        pull, bull, bush, push, butch, putsch
        Contre-ex. : clutch /klʌtʃ/
      • 5) "ASK words" : ‹a› + /f, s, θ/ → /ɑː/ (SSBE)
        staff, bath, class, glass, grass, pass, after, craft, ask, basket, mask, task, gasp
        Aussi : -ance#, -anch#, -and/t#, -ample# → advance, France, dance, branch, command, grant, plant, example
        GA : /æ/ (pas de changement)
      • 6) Règle de Luick : antépénultième accentuée → valeur entravée
        Si la voyelle monographe accentuée est en syllabe antépénultième (ou plus tôt), et suivie d'une seule consonne → valeur checked.
        • 6A) Alternances base ↔ dérivé
          sane /eɪ/ → sanity /æ/ ; sphere /ɪə/ → spherical /e/ ; bible /aɪ/ → biblical /ɪ/
        • 6B) Exceptions (isomorphisme fort)
          favor /eɪ/ → favorite /eɪ/ ; rare /eə/ → rarity /eə/ ; glory /ɔː/ → glorify /ɔː/ ; motive /əʊ/ → motivate /əʊ/
      • 7) Prononciations particulières de ‹o›
        • 7A) ‹o› → /ʌ/
          son, front, company, monk, month, sponge, tongue, wonder, worry, come, glove, love, other, brother
        • 7B) ‹o› → /uː/
          do, to, two, who, lose, move, prove, tomb, womb
    • Digraphes vocaliques
      Séquence de 2 lettres-voyelles = 1 phonème. 9 catégories principales.
      • 1) ‹ai›, ‹ay›, ‹ei›, ‹ey› → /eɪ/
        claim, straight, maid ; stay, gray ; eight, vein, neighbour ; convey, prey, grey
        💡 = valeur libre de ‹a› (Kate /eɪ/)
        Exc. : /aɪ/ aisle · /iː/ quay, conceive, seize, ceiling · /æ/ plaid · /e/ said, says
        • + ‹r› → /eə/
          hair, lair, affair ; heiress, theirs
          💡 = valeur libre+R de ‹a› (care /eə/)
          Exc. : /ɪə/ weird, eyrie
      • 2) ‹ea›, ‹ee›, ‹ie› (non final) → /iː/
        beast, meal, team ; feel, between, see ; piece, achieve, believe, siege
        💡 = valeur libre de ‹e› (Pete /iː/)
        Exc. ‹ea› : /e/ bread, breakfast, dead, head, heaven, measure, weather · /eə/ bear, pear, swear, wear · /ɜː/ earn, earth, heard, pearl, search
        • + ‹r› → /ɪə/
          appear, dear, hearing, shear ; leer, beer, engineer ; fierce, pierce
          💡 = valeur libre+R de ‹e› (here /ɪə/)
      • 3) ‹ie#›, ‹ye#› (en finale) → /aɪ/
        pie, die, lie, tie ; bye, Skye, lye, rye
        💡 ‹ie› non final = /iː/ (catégorie 2) vs ‹ie› final = /aɪ/ (catégorie 3). La position change tout.
      • 4) ‹oe#›, ‹oa› (non final) → /əʊ/
        doe, Joe, toe ; boast, coal, groan, roach
        💡 = valeur libre de ‹o› (note /əʊ/)
        Exc. : /ɔː/ broad, abroad · /ʌ/ does
        Cas particulier : ‹oe›/‹ae› gréco-latins → /iː/ en SSBE : amoeba, foetus, Caesar
        • ‹oa› + ‹r› → /ɔː/
          aboard, boar, hoard, roar, soar
          💡 = valeur libre+R de ‹o› (more /ɔː/)
      • 5) ‹eu›, ‹eau›, ‹ew›, ‹ue#› → /(j)uː/
        neural, euphemism, eulogy ; beauty ; stew, few, newt ; glue, due, blue
        💡 = valeur libre de ‹u› (mute /(j)uː/)
        Exc. : ‹eau#› → /əʊ/ dans emprunts français : plateau, beau, Bordeaux
        • ‹eu› + ‹r› → /(j)ʊə/ ou /(j)ɔː/
          Europe, Eurovision, Eurostar
          💡 = valeur libre+R de ‹u› (pure /(j)ʊə/)
          Exc. emprunts français ‹eu› + ‹r#› → /ɜː/ : entrepreneur, auteur, coiffeur
      • 6) ‹au›, ‹aw› → /ɔː/
        applause, cause, nausea ; hawk, gnaw, awkward
        Exc. : /ɒ/ because, laurel, sausage · /ɑː/ aunt, laugh, draught · /əʊ/ mauve · /aʊ/ gaucho, sauerkraut · /eɪ/ gauge
        • ‹au› + ‹r› → /ɔː/
          aural, dinosaur, thesaurus, aurora
      • 7) ‹oo› → /uː/
        bloom, bamboo, pool, spoon, moon, cartoon
        Exc. : /ʌ/ blood, flood · /ʊ/ foot, good, hood, wood, wool, cook, shook · /əʊ/ brooch
        • ‹oo› + ‹r› → /ɔː/
          door, floor, poor, moor
          poor, moor parfois /ʊə/, mais /ɔː/ largement dominant aujourd'hui.
      • 8) ‹oi›, ‹oy› → /ɔɪ/
        choice, appoint, toil ; boy, toy, oyster, employ
        • ‹oi› + ‹r› → rares, emprunts français /wɑː/
          abattoir, répertoire
          Exc. : /aɪə/ choir
      • 9) ‹ou›, ‹ow› → /aʊ/
        about, hound, foul, doubt, mouse ; browse, owl, cow, down, now
        Exc. : /ʌ/ cousin, couple, double, enough, touch, young · /əʊ/ although, boulder, soul, mould + après /m,n,l,r/ + ‹ow› : blow, grow, low, snow · /uː/ coup, route, you, youth · /ɔː/ bought, ought, thought · /ʊ/ could, should, would
        💡 Paires minimales : bow /baʊ/ (s'incliner) vs /bəʊ/ (arc) · row /raʊ/ (dispute) vs /rəʊ/ (ramer) · sow /saʊ/ (truie) vs /səʊ/ (semer)
        • ‹ou› + ‹r›
          → /ɔː/ : court, source, four, discourse
          → /aʊə/ : our, hour, flour, scour
          Exc. : /ɜː/ adjourn, courtesy, journey, journal
      📌 Points clés :
      1. Les valeurs de base des digraphes = souvent les valeurs libres des monographes correspondants.
      2. La modification par ‹r› → diphtongues centralisantes ou voyelles longues (même principe que les monographes).
      3. Les exceptions sont souvent liées à l'étymologie (emprunts français, grec, latin).
