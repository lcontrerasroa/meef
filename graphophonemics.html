1. Graphophonémique

Référentiel : Standard Southern British English (SSBE). Sauf mention contraire, les valeurs ci-dessous concernent des syllabes accentuées (marquées par « ʼ »).

Sous-règles principales et exceptions

1) <a> et <o> en finale <alC#> / <olC#>

  • <a> + l + (l#, t, d, k) → /ɔː/ : mall/mɔːl/, talk/tɔːk/, waltz/wɔːls/, bald/bɔːld/, falcon/ˈfɔːlkən/
  • <a> + l + (m/f) → /ɑː/ : half/hɑːf/, calm/kɑːm/, calf/kɑːf/, balm/bɑːm/
  • <o> + l + (l#, t, d, s, k) → /əʊ/ : roll/rəʊl/, bolt/bəʊlt/, told/təʊld/, yolk/jəʊk/, post/pəʊst/

NB : dans <-lk#>, le <l> est muet : walk/wɔːk/, talk/tɔːk/, folk/fəʊk/, yolk/jəʊk/.

2) « Règle du w » : /w/ + <a> et /w/ + <ar>

  • w + a → [wɒ] : water/ˈwɒtə/, want/wɒnt/, quality/ˈkwɒlɪti/, swan/swɒn/ (sauf si la consonne suivante est vélaire : quack/kwæk/, wax/wæks/, wag/wæg/)
  • w + ar → [wɔː] : warm/wɔːm/, swarm/swɔːm/, quarter/ˈkwɔːtə/

3) Finales particulières

  • <-ange#> : <a> → /eɪ/ : arrange/əˈreɪndʒ/, range/reɪndʒ/; aussi angel/ˈeɪndʒəl/, danger/ˈdeɪndʒə/
  • <-aste#> : <a> → /eɪ/ : paste/peɪst/, taste/teɪst/, waste/weɪst/
  • <-igh(t)#>, <-ign#>, <-ind#> : <i> → /aɪ/ : high/haɪ/, sign/saɪn/, find/faɪnd/; aussi child/tʃaɪld/, climb/klaɪm/, pint/paɪnt/

4) « Règle pull » : <f/p/b> + <u> + <-ll/-sh/-tch> → /ʊ/

pull/pʊl/, bush/bʊʃ/, butch/bʊtʃ/, putsch/pʊtʃ/

5) « ASK words » : <a> + /f, s, θ/ (+ occlusive sourde)

SSBE : /ɑː/ → staff/stɑːf/, bath/bɑːθ/, ask/ɑːsk/, basket/ˈbɑːskɪt/… (GA : /æ/)

Finales proches : <-ance#>, <-anch#>, <-and/t#>, <-ample#> → souvent /ɑː/ : advance/ədˈvɑːns/, France/frɑːns/, sample/ˈsɑːmpəl/

6) « Règle de Luick »

Quand une voyelle monographique est accentuée à l’antépénultième (ou plus tôt), suivie d’une seule consonne, elle prend sa valeur checked : character/ˈkærəktə/, federal/ˈfedərəl/, personality/ˌpɜːsəˈnælɪti/. Nombreuses exceptions existent et l’isomorphie lexicale peut primer : favour~favourite/ˈfeɪvə(r) ~ ˈfeɪv(ə)rɪt/.

7) Prononciations particulières de ‹o›

  • <o> → /ʌ/ : son/sʌn/, front/frʌnt/, company/ˈkʌmpəni/, wonder/ˈwʌndə/, love/lʌv/
  • <o> → /uː/ (mots fréquents) : do/duː/, to/tuː/, two/tuː/, who/huː/, lose/luːz/, move/muːv/

Voyelles inaccentuées

1) Règles générales

  • ‹a,e,o,u› → /ə/ : salad/ˈsæləd/, amulet/ˈæmjʊlət/, pilot/ˈpaɪlət/
  • ‹a,e,i› → /ɪ/ : cottage/ˈkɒtɪdʒ/, budget/ˈbʌdʒɪt/, stupid/ˈstjuːpɪd/
  • ‹u› → /(j)ʊ/ : educate/ˈedjʊkeɪt/, circular/ˈsɜːkjʊlə/, fabulous/ˈfæbjʊləs/

2) En pré-tonique

  • V + ‘V (syllabe suivante libre) → valeur libre : reality/riˈæləti/, zoology/zuːˈɒlədʒi/, iambic/aɪˈæmbɪk/
  • V + ‘C → valeur réduite régulière (/ə/ ou /ɪ/) : amend/əˈmend/, serene/səˈriːn/, atomic/əˈtɒmɪk/
  • V + CC → valeur checked : umbrella/ʌmˈbrelə/, dogmatic/dɒɡˈmætɪk/, partition/pɑːˈtɪʃən/

3) En finale V# / VC0 (<i,y#> et cas de diérèse)

  • ‹a› → /ə/ : sofa/ˈsəʊfə/, hyena/haɪˈiːnə/
  • ‹e,i/y› → /i/ : acne/ˈækni/, oceanic/ˌəʊʃiˈænɪk/, happy/ˈhæpi/
  • ‹u› → /u/ : influence/ˈɪnfluəns/, situation/ˌsɪtʃuˈeɪʃən/

Digrammes vocaliques (valeurs de base, variantes r et exceptions)

1) <ai>, <ay>, <ei>, <ey> → /eɪ/

claim/kleɪm/, day/deɪ/, vein/veɪn/, prey/preɪ/ …

+ <…r> → /eə/ : hair/heə/, their/ðeəz/

Exceptions fréquentes : aisle/aɪl/, either/ˈiːðə ~ ˈaɪðə/, quay/kiː/, said/says/sed, sez/, plaid/plæd/, surveillance/səˈveɪləns/.

2) <ea>, <ee>, <ie> (non final) → /iː/

beast/biːst/, see/siː/, believe/bɪˈliːv/, siege/siːdʒ/

+ <…r> → /ɪə/ : dear/dɪə/, engineer/ˌendʒɪˈnɪə/, pierce/pɪəs/

Exceptions : /e/ (le plus fréquent) head/hed/, bread/bred/, measure/ˈmeʒə/… ; /eə/ bear/beə/, /ɜː/ earth/ɜːθ/…

3) <ie#>, <ye#> (final) → /aɪ/

pie/paɪ/, tie/taɪ/, bye/baɪ/, rye/raɪ/

4) <oe#>, <oa> (non final) → /əʊ/

toe/təʊ/, boast/bəʊst/, coal/kəʊl/

+ <…r> → /ɔː/ : aboard/əˈbɔːd/, roar/rɔː/

Mots savants en <oe/ae> (latin/grec) : SSBE souvent /iː/ (foetus/ˈfiːtəs/, Caesar/ˈsiːzə/) ; exceptions /e/ (haemorrhage/ˈhemərɪdʒ/).

5) <eu>, <eau>, <ew>, <ue#> → /(j)uː/

euphemism/ˈjuːfəmɪzəm/, beauty/ˈbjuːti/, few/fjuː/, glue/ɡluː/

+ <…r> → /(j)ʊə ~ (j)ɔː/ : Europe/ˈjʊərəp ~ ˈjɔːrəp/  |  Exceptions FR : entrepreneur/ˌɒntrəprəˈnɜː/, coiffeur/kwɑːˈfɜː/.

Final <eau#> français → /əʊ/ : plateau/ˈplætəʊ/, Bordeaux/bɔːˈdəʊ/.

6) <au>, <aw> → /ɔː/

cause/kɔːz/, awkward/ˈɔːkwəd/

Exceptions : /ɒ/ because/bɪˈkɒz/ ; /ɑː/ laugh/lɑːf/ ; /əʊ/ mauve/məʊv/ ; /aʊ/ gaucho/ˈɡaʊtʃəʊ/ ; /eɪ/ gauge/ɡeɪdʒ/.

7) <oo>

→ /uː/ : moon/muːn/ ; → /ʊ/ : good/ɡʊd/ ; → /ʌ/ : blood/blʌd/ ; + <…r> → /ɔː/ : door/dɔː/ (variantes /ʊə/ rares aujourd’hui).

8) <oi>, <oy> → /ɔɪ/

choice/tʃɔɪs/, boy/bɔɪ/

Cas avec <…r> (emprunts FR) souvent /wɑː/ : abattoir/ˈæbətwɑː/, repertoire/ˈrepətwɑː/ ; exception : choir/ˈkwaɪə/.

9) <ou>, <ow> → /aʊ/

about/əˈbaʊt/, cow/kaʊ/, down/daʊn/

Exceptions : /ʌ/ young/jʌŋ/, touch/tʌtʃ/ ; /əʊ/ although/ɔːlˈðəʊ/, blow/bləʊ/ ; /uː/ route/ruːt ~ ruːt/, you/juː/ ; /ɔː/ thought/θɔːt/ ; /ʊ/ could/kʊd/.

Paires minimales selon le sens : bow/baʊ/ (v.) ~ /bəʊ/ (n.), row/raʊ/ (n.) ~ /rəʊ/ (n./v.), sow/saʊ/ (n.) ~ /səʊ/ (v.).
