    Hiérarchie des catégories morphologiques
    • non-affixés
      Lexèmes simples
    • affixés
      Suffixes neutres et contraignants
      • suffixes neutres
        Ne déplacent pas l’accent
        • flexionnels
          Exemples : ˈwalks, ˈtalked, ˈcats
        • dérivationnels
          Exemples : ˈteacher, ˈkingdom, ˈhappiness
      • suffixes contraignants
        Imposent un déplacement de l’accent
        • qui placent l’accent avant eux-mêmes
          • ic
            Ex. : geˈneric, aˈtomic, neoˈlithic, ˈethics
          • ify/efy
            Ex. : ˈterrify, ˈqualify, inˈdemnify, ˈstupefy
          • ity/ety
            Ex. : vaˈriety, anˈxiety, moˈbility, priˈority
          • ible
            Ex. : ˈfeasible, adˈmissible, ˈsensible, ˈgullible
          • igible
            Ex. : ˈcorrigible, ˈdirigible, inˈtelligible
          • train de voyelles
            • e+V+(C)
              Ex. : ˈcereal, eˈthereal, ˈarea, ˈapnea, ˈnausea
            • i+V+(C)
              Ex. : ˈperiod, ˈSerbia, maˈterial, ˈmartian, peˈculiar
            • u+V+(C)
              Ex. : ˈsensual, Nicaˈragua, ˈQuechua, garˈgantuan, ˈusual
        • dynamiques
          • ATE
            • 2 syllabes
              • verbe (01 et /eɪt/)
                Ex. : loˈcate, doˈnate, reˈlate
              • non-verbe (10 et /ət/)
                Ex. : ˈsenate, ˈcurate
            • plus de 2 syllabes
              • verbe (-100 et /eɪt/)
                Ex. : ˈdefecate, eˈnunciate, ˈintegrate
              • non-verbe (-100 et /ət/)
                Ex. : ˈmoderate, ˈdelicate, ˈobstinate, ˈultimate
              Rappel : -ate verbal → /eɪt/; adjectival/nominal → /ət/ avec accent initial.
          • ATIVE
            Il y a 3 cas possibles du suffixe -ative. Attention aux trissyllabes en -ative qui prennent souvent /-100/.
            • 0. ⚪ -IVE (similaire) est neutre
              Ex. : aˈbuse → aˈbusive /010/; supˈport → supˈportive /010/; imˈpulse → imˈpulsive /010/; ˈtribute → attˈributive /0100/; corˈrode → corˈrosive /010/; exˈplode → exˈplosive /010/; desˈcribe → desˈcriptive /010/; eˈmotion → eˈmotive /010/; comˈpete → comˈpetitive /0100/; diˈminish → diˈminutive /0100/.
              ❗ Exception : ˈexecute → eˈxecutive /0100/.
            • 1. 🟢 -ATIVE neutre
              Base + -ative : neutre
              Ex. : ˈtalk → ˈtalkative /100/; proˈvoke → proˈvocative /0100/; deˈclare → deˈclarative /0100/; inˈterpret → inˈterpretative /01000/; auˈthority → auˈthoritative /01000/; iˈmagine → iˈmaginative /0100/.
              Verbe en -ate (≥ 3 syllabes) + -ive : neutre → acˈcelerate → acˈcelerative /01000/; ˈdecorate → ˈdecorative /1000/; ˈeducate → ˈeducative /1000/; eˈlaborate → eˈlaborative /01000/; ˈcumulate → ˈcumulative /1000/.
              ❗ Exceptions : ˈdemonstrate → deˈmonstrative /0100/; ˈindicate → inˈdicative /0100/; ˈalternate (vb) → alˈternative /0100/; inˈterrogate → ˌinterˈrogative /00100/.
            • 2. Trissyllabes en -ATIVE → /-100/ (souvent)
              Base monosyllabique + -ative → /-100/ (neutre)
              talk → ˈtalkative /100/; cause → ˈcausative /100/; cure → ˈcurative /100/.

              Aucun dérivant correspondant → /-100/ (contraignant)
              ˈlucrative /100/; noms liés : ˈfricative /100/, ˈadjective /100/, ˈtentative /100/.

              Verbe en -ate accentué /01/ + -ive → /100/ (contraignant)
              reˈlate → ˈrelative /100/; neˈgate → ˈnegative /100/; narˈrate → ˈnarrative /100/; seˈdate → ˈsedative /100/; loˈcate → ˈlocative /100/.
              🟢 Exception notable : creˈate → creˈative /010/.
          • ARILY
            Rappel : -ly est neutre. C’est la séquence -arily qui crée un schéma /-100/ en anglais américain.
            • 1) -LY neutre
              happy /10/ → ˈhappily /100/; interesting /100/ → interestingly /1000/; laborious /0100/ → laboriously /01000/.
            • 2) Anglais britannique
              Neutre : iˈmaginary /01000/ → iˈmaginarily /010000/; heˈreditary /01000/ → heˈreditarily /010000/; ˈliterary /1000/ → ˈliterarily /10000/.
              Réduction courante /ərəli/, parfois une syllabe tombe : ˈlɪtrəri /100/ → ˈlɪtrərili /1000/.
            • 3) Anglais américain
              Impose /-100/ : ˈnecessary /1000/ → ˌnecesˈsarily /20100/; eleˈmentary /20100/ → ˌelemenˈtarily /200100/; ˈordinary /1000/ → ˌordiˈnarily /20100/.
              Réalisation phonétique typique : /ˈerili/.
              En GB, certaines variantes avec /ˈeərili/ existent aussi : ˌarbitˈrarily /20100/, ˌnecesˈsarily /20100/, priˈmarily /0100/, ˌvolunˈtarily /20100/.
            Attention : -ly est neutre.
            Ne pas dire que -ly impose /-100/. C’est la séquence -arily (et surtout en anglais américain) qui entraîne /-100/.
            Ex. : ˈhappy → ˈhappily /100/; comˈpetitive → comˈpetitively /01000/; ˈagonizing → ˈagonizingly /10000/.
          • terminaisons adjectivales
            En général neutres : -al, -ar, -ous, -ant, -ent, -y. Un agrégat consonantique avant le suffixe favorise -010.
            • avec agrégat consonantique (-010)
              inˈsistent; conˈtemptuous; coˈlossal; beˈneficent; aˈbusive
            • sans agrégat consonantique (-100)
              ˈdangerous; ˈfamous; ˈcurious; ˈelegant; ˈarrogant; ˈgeneral; ˈanimal
        • qui prennent l’accent sur eux-mêmes
          • -ADE
            greˈnade, paˈrade, paliˈsade, sereˈnade → emprunts au français; lemoˈnade, oranˈgeade, liˈmeade…
            Exceptions : ˈdecade, ˈrenegade, ˈmarmalade.
            Verbes en -vade/-grade : inˈvade, perˈvade, deˈgrade → accent final pour d’autres raisons (pas des emprunts).
          • -AIR(E)
            millioˈnaire, billioˈnaire, doctriˈnaire, questionˈnaire, deboˈnair, Aˈpolliˌnaire → emprunts au français.
          • -ESE
            Chiˈnese, Congoˈlese, Lebaˈnese, portuˈguese.
          • -QUE / -ESQUE
            anˈtique, baˈroque, maˈcaque, Mozamˈbique, araˈbesque; humoˈresque, statuˈesque, Ramboˈesque, Kafkaˈesque.
            Attention à l’emplacement syllabique de l’accent : anˈtique /01/ (pas *antiˈque*).
          • -ESCENCE / -ESCENT
            Schéma /-10/ : adoˈlescence, evaˈnescence, incanˈdescence, recruˈdescence; adoˈlescent, efferˈvescent, obsoˈlescent, phosphoˈrescent, remiˈniscent.
          • <-VˈV'(C)(e)> : doubles voyelles identiques
            Suffixe -ee et autres finales à double voyelle tonique
            traiˈnee, refuˈgee, nomiˈnee, interviewˈee; deˈgree, chimˈpanzee, jamboˈree; bamˈboo, taˈboo, kangaˈroo, cockaˈtoo; baˈzaar, genˈteel, eighˈteen, engiˈneer, baˈboon, Cameˈroon; aussi caˈboose, paˈpoose.

            ✨ Les rares mots en -uum ne sont pas concernés : conˈtinuum, reˈsiduum, ˈvacuum (ici <uu> = deux voyelles distinctes).
            Exceptions : ˈcuckoo, ˈigloo, ˈLiverpool, ˈbaboo, ˈvoodoo, ˈcoffee, ˈcommittee, ˈtoffee, ˈyankee, ˈapogee, ˈspondee, ˈtrochee, ˈtepee.
            Ne pas confondre avec des composés morphologiques : ˈfalsehood, ˈgreenroom, ˈsunscreen.
          • <-CˈC'e> : doubles consonnes identiques + <e>
            giˈraffe, aquaˈrelle, queˈnelle, juliˈenne, laˈcrosse, laundeˈrette.
            Non concernés : composés en -gramme (ˈkilogramme…).
            Ne pas assimiler : ˈvoiceless, ˈkindness; noms féminins en -ess (ˈwaitress, ˈgoddess, ˈshepherdess, ˈauthoress…).
          • classiques attracteurs (rappel)
            -ee : employˈee, trainˈee; -eer : enginˈeer, voluntˈeer; -ese : Japanˈese; -esque : Romanˈesque; -ette : cigarˈette; -ique : technˈique; -ade : parˈade; -oon : ballˈoon.
